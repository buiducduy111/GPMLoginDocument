# Đóng profile

API URL

```
GET: /api/v3/profiles/close
```

Params:

| Tên param | Bắt buộc | Mô tả          |
| --------- | -------- | -------------- |
| id        | Có       | ID của profile |

Ví dụ

```
http://127.0.0.1:19995/api/v3/profiles/close?id=xgyasg1995
```

Repsonse

```json
{
    "success": true,
    "message": "Đóng thành công"
}
```
