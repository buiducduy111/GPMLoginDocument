**Read Json**

* Đầu vào: định dạng Json;node 1;node 2;node 3....

```
Ví dụ có một json như sau được lưu vào biến $ketQua

{
    "name": "Test profile",
    "info": {
        "created_at": "2023-11-11",
        "author": "buiducduygame"
    }
}

Để đọc được author dùng như sau:
$ketQua;info;author
```
